# SaaS UI Overhaul — 设计文档

**日期**: 2026-05-29
**状态**: 待实现
**分支**: `saas-ui-overhaul`
**设计风**: 靛蓝紫玻璃 Indigo Glass — 双模式

## 目标

全系统 UI 从移动端延伸风格改造为专业 SaaS 企业后台风格。参考 Linear / Vercel / Stripe 布局模式 + Apple 玻璃质感。

## 配色方案

| 元素 | 颜色 | 说明 |
|------|------|------|

| 侧边栏 | `rgba(255,255,255,0.48)` + backdrop-blur | 毛玻璃半透明 |
| KPI卡片 | `rgba(255,255,255,0.48)` + 微边框 + inset高光 | 毛玻璃卡片 |

| 强调色 | `#4F46E5` | 靛蓝紫 |
| 强调色浅 | `#818CF8` | hover/激活态 |
| 文字主 | `#1E1E3D` | 深蓝黑 |
| 文字辅 | `#8B8BB0` | 灰蓝 |
| Tab激活 | `rgba(79,70,229,0.08)` 底 + 蓝紫文字 | 淡彩高亮 |

## 组件改造

### 1. 侧边栏 `ResponsiveScaffold`

| 当前 | 改造后 |
|------|--------|
| 56px 纯图标 | 220px 图标+文字横向排列 |
| 无分组 | MAIN / BUSINESS 两个分组，组间分割线 |
| collapsed 模式 56px | 保持折叠能力，折叠后仅图标 |
| 无当前模块高亮 | 当前模块 `accent bg + colored text` |

### 2. 全局主题 `AppTheme`

- 去掉所有渐变 KPI 卡片（紫色/粉色/蓝色/绿色）
- KPI 卡片改为单色 + 微阴影
- Light: 玻璃模糊 + 半透明 + 内高光
- Dark: 纯底色 + 细边框
- 统一用靛蓝紫 #4F46E5 作为 accent
- 字体保持 Microsoft YaHei

### 3. 页面内 Tab 替代全屏切换

所有模块 Dashboard（财务/HR/市场/招投标/项目管理）:
- 面包屑导航：`首页 › 模块名`
- TabBar 切换子页面（Dashboard / 票据 / 预算 / 支出 / 凭证）
- 去掉 Navigator.push 全屏切换
- 去掉子页面的 AppBar 返回按钮

### 4. 数据表格化

以下列表从卡片堆叠改为紧凑 DataTable：
- 票据列表
- 支出列表
- 凭证列表
- 客户列表
- 项目列表

### 5. Dashboard 适配

财务 Dashboard 适配新配色：
- KPI 卡片：玻璃质感（Light）/ 纯色细边框（Dark）
- 收入折线图：靛蓝紫线
- 预算进度条：分类色

## 实施优先级

1. 主题配色（AppTheme 重写）— 最先，所有后续依赖
2. 侧边栏重构（ResponsiveScaffold）— 全局布局基础
3. Dashboard 适配（财务）— KPI卡片重做
4. 页面内 Tab（财务模块）— 去掉全屏切换
5. 数据表格化（票据页）— 卡片→DataTable
6. 扩展到其他模块 — 市场/招投标/PM/HR

## 不改变

- 后端 API — 零改动
- 业务逻辑 — 零改动  
- 数据模型 — 零改动
- 文件预览系统 — 不变
