Apple 玻璃风特点

核心元素

  • 微渐变底（暖灰/深蓝）替代纯平底色
  • 毛玻璃卡片：backdrop-blur + 半透明 + 细边框
  • 内阴影高光：inset 0 1px 1px rgba(white, 0.8)
  • 微妙圆角：8-10px，不过度
  • 极克制阴影：0 1px 2px 几乎感觉不到
  • 统一蓝色强调 #3B82F6

在 Flutter 里的实现

  • BackdropFilter + ClipRect 替代 CSS blur
  • Container gradient 做微渐变背景
  • BoxDecoration border 0.5px 细线
  • BoxShadow 替代 CSS shadow
  • 用 withAlpha 做半透明层

选配色方向

A

暖灰玻璃（浅色）

暖米灰渐变底 + 白色半透明卡片 + 内高光 + 蓝色强调

B

深蓝玻璃（深色）

深蓝灰渐变底 + 暗色半透明卡片 + 微光边框 + 蓝色强调

C

两者都要

浅色/深色双模式，都走玻璃质感，自动跟随系统主题
